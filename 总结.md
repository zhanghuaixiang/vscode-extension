## 总结

## 1、如何在左侧添加新tab展示栏？
### package.json中配置
在`contributes`下的`viewsContainers`和`views`中新增配置。
```
"contributes": {
    "viewsContainers": {
        "activitybar": [
            {
                "id": "conventional-review",
                "title": "review提交",
                "icon": "images/icon.svg"
            }
        ]
    },
    "views": {
        "conventional-review": [
            {
                "id": "review_1",
                "name": "第一个"
            },
            {
                "id": "review_2",
                "name": "第2个"
            }
        ]
    }
}
```

## 2、如何在顶部弹出可供选择的输入框？
通过`vscode.window.createQuickPick()`，返回个`quickPick`对象，可对此对象进行属性修改，如step、totalStep、items、buttons等。
